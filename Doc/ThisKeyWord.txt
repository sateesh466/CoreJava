This KeyWord

this is a keyword in Java. 
It can be used inside the method or constructor of a class. It(this) works as a reference to the current object, 
whose method or constructor is being invoked. 
This keyword can refer to any member of the current object from within an instance method or a constructor.


Usage of java this keyword


this can be used to refer current class instance variable.
this can be used to invoke current class method (implicitly)
this() can be used to invoke current class constructor.
this can be passed as an argument in the method call.
this can be passed as argument in the constructor call.
this can be used to return the current class instance from the method.
